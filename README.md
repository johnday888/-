## 🚀 快速开始
1. 从 [Releases](../../releases) 页面下载最新版的 `love.exe`。
2. 准备一首背景音乐并命名为 `love.mp3`， 放在love.exe同一个目录下 。
3. 双击运行，感受浪漫。

## 🛠️ 技术栈
* Python 3.14
* Pygame-ce (音频驱动)
* Tkinter (动画渲染)

##  注意事项
1. 由于版权原因，本仓库不提供音乐文件  
2. 如需自行生成exe，可自行准备ico图标并放在项目根目录下
